
# e2e-python-cli -- Skill Summary

## Overview
This plugin provides bundled Python scripts designed for end-to-end testing and utility operations. It offers a simple way to execute Python-based commands directly through the skill interface, making it useful for testing workflows and performing basic scripting tasks.

## Usage
Ensure Python 3 is installed on your system, then execute the bundled scripts using the provided commands. The plugin includes pre-flight checks to verify Python availability.

## Commands
| Command | Usage | Purpose |
|---------|-------|---------|
| Hello | `python3 scripts/hello.py` | Test the e2e-python-cli functionality and output a greeting message |

## Triggers
An AI agent should activate this skill when needing to test Python script execution, verify end-to-end functionality, or demonstrate basic Python plugin capabilities. Use it for validation and testing scenarios.
