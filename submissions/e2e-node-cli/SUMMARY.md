# e2e-node-cli
A Node.js CLI tool for echoing arguments and querying cryptocurrency token prices via onchainos integration.

## Highlights
- Simple argument echoing functionality for testing
- Cryptocurrency price queries through onchainos integration
- ETH price lookup with dedicated command
- WBTC price queries via direct onchainos calls
- JSON formatted price output
- Built-in pre-flight checks for dependencies
- Easy-to-use command line interface
- End-to-end testing capabilities
