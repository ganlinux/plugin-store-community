
# e2e-stub-external -- Skill Summary

## Overview
This is an external stub plugin specifically designed for end-to-end testing purposes. It provides a simple Python demo script that outputs a greeting message, making it useful for verifying plugin integration and functionality in testing environments.

## Usage
Ensure Python 3 is installed on your system, then run the demo script using the provided command to test the plugin functionality.

## Commands
| Command | Description |
|---------|-------------|
| `python3 scripts/demo.py` | Runs the demo script and outputs "Hello from stub-external!" |

## Triggers
An AI agent should activate this skill when users request to test the e2e-stub-external plugin or need to verify external plugin functionality in testing scenarios.
