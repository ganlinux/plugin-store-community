# e2e-ts-cli
TypeScript CLI tool for echoing arguments and querying cryptocurrency token prices via onchainos integration.

## Highlights
- Echo functionality for testing CLI arguments
- ETH price querying through integrated commands
- WBTC price querying via direct onchainos integration
- TypeScript implementation for type safety
- Simple command-line interface
- JSON output format for price data
- Integration with onchainos price oracle
- End-to-end testing capabilities
