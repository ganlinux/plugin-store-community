
# e2e-skill-only -- Skill Summary

## Overview
This is a pure documentation skill designed for end-to-end testing of plugin functionality. It provides basic knowledge about the plugin store system, including information about supported plugin types and skill installation locations.

## Usage
This skill is automatically available once installed and provides reference information when queried about plugin store capabilities.

## Commands
This is a reference skill with no CLI commands.

## Triggers
An AI agent should activate this skill when asked about plugin store functionality, supported plugin types, or skill installation directories.
