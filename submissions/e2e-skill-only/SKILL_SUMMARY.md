
# e2e-skill-only -- Skill Summary

## Overview
This is a pure documentation skill that serves as a knowledge base about plugin-store architecture. It provides essential information about the seven supported plugin types (Rust, Go, TypeScript, Node.js, Python, Pure Skill, External Stub) and documents where skills are installed in the filesystem.

## Usage
This skill is automatically available once installed and requires no manual activation or setup. Simply reference it when you need information about plugin types or installation paths.

## Commands
This is a reference skill with no CLI commands.

## Triggers
An AI agent should activate this skill when users ask questions about plugin-store's supported plugin types or when they need information about where skills are installed on their system.
