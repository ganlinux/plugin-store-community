# e2e-skill-only
A pure documentation skill with no binaries or scripts that provides knowledge about plugin types and installation paths.

## Highlights
- Pure documentation-based skill with no executable components
- Provides knowledge about plugin-store's 7 supported plugin types
- Contains information about skill installation locations
- Serves as an E2E test reference for skill-only plugins
- Demonstrates minimal skill structure without binaries or scripts
- Educational resource for understanding plugin architecture
- Lightweight utility category plugin
- MIT licensed open-source tool
